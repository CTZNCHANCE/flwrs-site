# FLWRS — Recognition, In Real Time

A cultural recognition platform and IRL experience created and hosted by **O.N.E The Duo** (Tekitha & Prana Supreme).

## About

FLWRS honors pioneers, builders, and visionaries while they are actively shaping culture — through intimate, long-form conversations rooted in impact, leadership, and legacy. Incubated and featured on platforms including Sway in the Morning on SiriusXM.

## Project Structure

```
flwrs-site/
├── public/
│   ├── index.html          # Main page
│   ├── css/
│   │   └── style.css       # Styles
│   ├── js/
│   │   └── main.js         # Interactions & ambient audio
│   └── images/             # All image assets
├── vercel.json             # Vercel deployment config
├── package.json
└── README.md
```

## Local Development

```bash
npm run dev
```

Serves the site at `http://localhost:3000`.

## Deployment

### Vercel (recommended)

1. Push this repo to GitHub
2. Import the repo in [Vercel](https://vercel.com)
3. Vercel auto-detects the static site config from `vercel.json`
4. Deploy

No build step required — this is a static site served directly from `/public`.
